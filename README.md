# Ykfastlane

iOS 通用打包工具的终端门户工具

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ykfastlane'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install ykfastlane

## Usage

安装之后，使用 ykfastlane help 查看使用说明
